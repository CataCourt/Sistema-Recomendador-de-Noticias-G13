# Proyecto de Sistema Recomendador de Noticias
## Integrantes: Marco Agüero, Catalina Court Y María de los Ángeles Ronfeldt.
Este consiste en un archivo Jupiter que contiene los datos a utilizar del dataset MIND, con sus celdas ejecutadas. En este archivo se descarga automáticamente los datasets a ocupar
Para su ehecución debe ser abierto en Colab y ejecutar todas la celdas, tener en cuenta que existen celdas de 30 minutos de ejecución, por lo que se debe tener paciencia en las espera de resultados.
